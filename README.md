# Tripleten web_project_homeland

## De Pátria para Pátria

## Descrição

O projeto consistiu na criação de uma página web responsiva que apresenta as cidades natais de alguns funcionários da TripleTen. A página possui seções como um cabeçalho com identidade visual, uma seção principal com título, subtítulo e uma imagem, uma introdução com texto e citação, e um rodapé.

O foco do projeto foi desenvolver o layout de uma pagina web com base no design do Figma, também garantindo uma melhor adaptação para diferentes tamanhos de tela. Foram utilizados conceitos de HTML e CSS, metodologia BEM, organização de arquivos em blocos separados,Flexbox e Media Queries.

## Tecnologias Utilizadas

- HTML
- CSS
- Metodologia BEM
- Flexbox
- Media Queries

## Ideias/Planos de Melhorias

- Adicionar mais seções ao site, como uma seção contendo varias imagens e descrições das cidades natais.
- Incluir links de navegação para facilitar a visualização de determinadas seções.
